Структура каталогов
-------------------

*application*::
	В этом каталоге находится ваше приложение, константа +APPPATH+ указывает на эту директорию и может быть изменена в +index.php+.
	Внутри вы размещаете каталоги модулей и конфиги приложения
	default.php;;
		В этом файле находятся основные настройки сайта, применяемые вне зависимости от того, на каком хосту он запущен
	<HTTP_HOST>.php;;
		Здесь находятся настройки, специфичные для конкретного хоста, например подключение к тестовой/локальной базе данных или proxy для корректной работы Curl
	_common/;;
		папка для общих моделей, слотов, тегов, и прочих ресурсов всего сайта
	*default/*;;
		Здесь находится основной модуль на который передается управление
		controllers/:::
			В папке находятся контроллеры, на которые передается управление
			[...]Controller.php::::
				файлы контроллеров
			IndexController.php::::
				основной контроллер - передается управление, если больше некуда
		templates/:::
			Директория шаблонов для данного модуля
			404.html::::
				Показывает страницу 404 для этого модуля (если нету, то смотрит в +default+)
			main.html::::
				Основной шаблон страницы по умолчанию
	[...]/;;
		директории модулей совпадающие по структуре с каталогом +default+
*cron*::
	Скрипты для запуска не через http
	cruise.php;;
		Демон для запуска в фоне действий через указанные промежутки времени
	start.php;;
		запуск одиночного действия с суффиксом Cli (например +php cron/start.php test/test/15+)
*doc*::
	Всякая документация
	asciidoc;;
		*То, что вы сейчас читаете* :)
*lib*::
	Папка с различными дополнительными библиотеками
	*Не готово*
*log*::
	Папка для сохранения логов по умолчанию (должна быть разрешена запись)
*QFW*::
	Общие классы фреймворка
	*QuickFW*;;
		Ядро
		Cacher:::
			Проксирующие классы кешеров из +Dklab_Cache+
		Auth.php:::
			Класс авторизации
		Cache.php:::
			Интерфейс загрузки кешеров
		Error.php:::
			Очень сырой обработчик ошибок :)
		Plugs.php:::
			Плагины к шаблонизаторам
		Router.php:::
			Роутер
		Session.php:::
			Класс для работы с сессиями
	Cacher;;
		Кешеры
	Templater;;
		Шаблонизаторы
	Init.php;;
		Файл начальной инициализации фреймворка
	*config.php*;;
		Настройки по умолчанию - подключаются парвыми
	license;;
		Лицензия на фреймворк :)

*tests*::
	Тесты для PHPUnit
*tmp*::
	Папка для временных файлов (должна быть разрешена запись)
	cache;;
		Файловый кеш
*www*::
	DOCUMENT ROOT
